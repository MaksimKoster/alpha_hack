2. **Перейдите в директорию `ensemble_2`:**

3. **Перейдите в директорию `ensemble_3`:**

### Шаг 2: Объединение прогнозов

1. **Перейдите в директорию `blending_submissions`:**
   - Откройте `*.ipynb`.
